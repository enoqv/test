# Dynamic allowlist for the Squid proxy.
#
# Syntax (squid dstdomain ACL):
#   - `.example.com`  matches `example.com` and ALL subdomains
#   - `example.com`   matches `example.com` exactly (no subdomains)
#   - never list both forms of the same domain — squid treats that as
#     a FATAL config error (the leading-dot form already covers the
#     bare form, see RemoveACL warnings in squid's cache.log)
#
# After editing, squid auto-reloads via the inotify watcher. To force a
# reload manually run `sudo reload-squid.sh` from inside the dev container.
